
## How To Update DLL Files

#### To update the DLL files:
- first download the ```"RDLN-UpdatedDLLs.zip"``` file above.
- Next, navigate to the folder where the ```"RedLineExecutor.exe"``` file is located.
- After that, drag **ALL** of the files from the zip file which you downloaded into that folder. No problem, I've already arranged the locations for all of them for you.
